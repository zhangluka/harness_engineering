### 🛠️ 手把手教程：3 分钟领取免费 API Key

#### 第一步：注册 NVIDIA 账号

1. 打开官网：**https://build.nvidia.com**
2. 点击右上角的「Login」按钮
3. 如果还没有账号，选择「Create Account」，填写：
   - 邮箱地址（推荐用 QQ 鈮箱 / Outlook，国内收验证码更稳）
   - 设置密码
4. 点击确认后，去邮箱查收验证邮件，点击链接完成激活

---

#### 第二步：验证手机号，解锁 API 权限

账号注册完成后，页面顶部会出现一个提示：**「Verify your account to unlock API access」**

1. 点击「Verify」
2. 输入你的手机号（国内手机号可以直接用，+86 开头）
3. 接收验证码，填入后提交

✅ 验证完成后，你的账号就解锁了完整的 API 调用权限

---

#### 第三步：创建你的 API Key

验证完成后，点击右上角的头像图标，在下拉菜单中选择「API Keys」

1. 进入 API Keys 管理页面，点击右上角的「**+ Create API Key**」按钮
2. 在弹出的对话框中填写：
   - **Name**：随便填一个名字，比如 `my-free-key`
   - **Expiration**：有效期 —— 这里可以选最长 12 个月（1 年）
3. 点击「Create Key」确认

⚠️ **重要提示**：API Key 创建成功后只显示一次！页面上会出现一串 `nvapi-` 开头的字符串，务必立刻复制保存，关闭弹窗后就看不到完整 Key 了。
