# AGENTS.md

这是智能体的入口文件，帮助智能体理解项目结构和协作方式。

## 目录结构

- `concepts/` — 核心概念笔记（稳定的核心定义）
- `thinking/` — 演进思考（动态的路线图、方向规划）
- `projects/` — 实战项目（主要工作区）
- `tools/` — 沉淀的工具
- `templates/` — 可复用模板
- `notes/` — 过程笔记


## 协作指南

当你需要在这个项目中添加内容时，请遵循以下原则：

1. **核心概念** → 放入 `concepts/`，确保内容稳定、经过验证
2. **演进思考** → 放入 `thinking/`，用于动态规划、路线图、方向探索
3. **实战项目** → 放入 `projects/`，每个项目一个目录，包含 README、harness、src、feedback
4. **工具沉淀** → 放入 `tools/`，记录可复用的工具/框架
5. **模板积累** → 放入 `templates/`，提供可复用的模板
6. **过程笔记** → 放入 `notes/`，记录过程中的思考、学习、踩坑
